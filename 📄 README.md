# 🧠 OLAP_CERT: MITRE ATT&CK ↔ CAPEC 매핑 프로젝트

이 프로젝트는 MITRE의 **ATT&CK 기술(Technique)** 과  
**CAPEC 공격 패턴(Attack Pattern)** 간의 관계를 자동으로 매핑합니다.  

보안 분석, 위협 헌팅, 공격 시나리오 자동화에 활용할 수 있습니다.
